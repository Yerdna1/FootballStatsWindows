üîê Authentication

Most endpoints require authentication using Firebase ID tokens. Include your token in the Authorization header:

Authorization: Bearer <your-firebase-id-token>

User Roles

  • User: Access to public data and personal profile
  • Moderator: Can manage leagues, teams, and fixtures
  • Admin: Full system access including user management

üì± Rate Limiting

To ensure fair usage and optimal performance, our API implements rate limiting:

  • General API: 60 requests per minute
  • Intensive operations: 10 requests per minute
  • Authentication: 5 requests per 15 minutes

Rate limit information is included in response headers.
