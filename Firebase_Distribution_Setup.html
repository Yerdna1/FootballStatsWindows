Firebase App Distribution Setup Guide

This guide will help you set up Firebase App Distribution to distribute your Football Stats installer to testers.

Note: Firebase App Distribution is primarily designed for mobile apps, but we can adapt it for Windows applications.

Prerequisites

  • Firebase project
  • Node.js and npm installed
  • Firebase CLI (will be installed in the steps below)

Step 1: Install Firebase CLI

Open a terminal and run:

npm install -g firebase-tools

Step 2: Login to Firebase

Authenticate with Firebase using your Google account:

firebase login

This will open a browser window for authentication.

Step 3: Set Up Your Firebase Project

  1. Go to the Firebase Console
  2. Create a new project or select an existing one
  3. In the left sidebar, click on "App Distribution"
  4. Follow the setup process to create a new app

Step 4: Prepare Distribution Manifest

Create a file named firebase-distribution-manifest.json with the following content:

{
  "version": "1.0.0",
  "releaseNotes": "Football Stats Installer - Initial Release\n\n• Custom football icon\n• Fixed Firebase authentication\n• Improved error handling"
}

Step 5: Distribute Your Installer

Run the following command to upload your installer:

firebase appdistribution:distribute FootballStats_Setup.exe --app YOUR_PROJECT_ID --release-notes-file firebase-distribution-manifest.json --groups testers

Replace YOUR_PROJECT_ID with your Firebase project ID.

Step 6: Invite Testers

  1. In the Firebase Console, go to App Distribution
  2. Find your app and click on it
  3. Click on "Testers & Groups"
  4. Add testers by email address

The testers will receive an email invitation to download the installer.

Tip: If you prefer a simpler approach, consider using Firebase Storage instead. Upload your installer to Firebase Storage, set the file permissions to public, and share the download URL.

For more detailed information, visit the Firebase App Distribution documentation.
